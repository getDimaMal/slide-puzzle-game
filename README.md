# slide-puzzle-game

## Setting Up

### To `run` project locally

```bash
  yarn install
  yarn run dev
```

### To `build` project

```bash
    yarn install
    yarn run build
```

### Codebase Commands

* `yarn run format`
* `yarn run lint:scripts`
* `yarn run lint:styles`
* `yarn run type-check`
* `yarn run test`
